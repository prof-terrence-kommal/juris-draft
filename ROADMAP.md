- TypeScript CLI for model-agnostic drafting packets.
- Jurisdiction profile loader.
- Tone profile loader.
- Markdown recipe system.
- JSON facts input.
- Markdown output.

## 0.4 Community

- Contributor profile checklist.
- Review labels for jurisdiction profiles.
- Examples for access-to-justice workflows.
