### NDA Outline Structure

1. Identify disclosing and receiving parties.
2. Define confidential information.
3. Define permitted purpose.
4. Set confidentiality obligations.
5. List exclusions from confidentiality.
6. Address return or destruction of information.
7. Add duration, governing law, and dispute resolution placeholders.
